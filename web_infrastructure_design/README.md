# Web Infrastructure Design

This project contains diagrams and explanations of different web infrastructure designs.

## Task 0 - Simple web stack

A simple one-server web infrastructure hosting `www.foobar.com`.

Components:
- Domain name: `foobar.com`
- DNS A record: `www` points to `8.8.8.8`
- One server
- Nginx web server
- Application server
- Application files
- MySQL database
